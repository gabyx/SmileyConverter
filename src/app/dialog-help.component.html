The image URL needs to be copied from an uploaded image to imgur.com/upload It should not be to big! e.g. 100px x 100px. You can resize the image on imgur directly!

For imgur.com image links you can add the following letters

  • h : Huge
  • l : Large
  • m : Medium
  • t : Tiny
  • b : Extra Tiny
  • s : Super Tiny

to the filename 78YFb1H.png e.g.: https://i.imgur.com/78YFb1Hs.png for a super small image.
